Analytical Lab

An overview of the Analysis Module. This module is used to store the results of plant material analyses.

📄️ Structure

A new structure of dataflow is (being) developed for the analytical lab. The basic structure is depicted below:

📄️ Analysis Order

AnalysisOrder table contains orders for analysis. This includes orders for Iribov Analytical services (Analytical Lab) as well as outgoing analysis orders to NAK-tuinbouw (and others). The aim for AnalysisOrder is to contain information about Who wants to Analyse with What aim and When.

📄️ Analysis Suborder

AnalysisSuborder is used as a grouping of AnalysisSamples, and a more detailed explanation of an AnalysisOrder.

📄️ Analysis Sample

AnalysisSamples are the individual (leaf) samples present in an AnalysisOrder. These samples are grouped together in an AnalysisSubOrder

📄️ Analysis Lysis

Analysis Lysis represents an individual lysis from an AnalysisSample. When a sample is prepared again, this would be a different Analysis Lysis. Analysis Lyses are usually put in a 96 well plate. When a Lysis record is put into a Analysis Lysis Plate, in the background only a connection is made by storing the Lysis Plate Name in the Lysis record. Additionally, the position of the lysis in the lysis plate is stored in the lysis record.

📄️ Analysis Isolation

AnalysisIsolation represents an individual isolation from an Analysis Lysis. When a lysis is isolated again, this would be a different AnalysisIsolation. AnalysisIsolations are usually put in a 96 well plate. This is tracked in the AnalysisIsolationPlate table.

📄️ Analysis Datapoint

Analysis Datapoint is the representation of the analysis performed after isolation. A Datapoint is therefore ALWAYS linked to an AnalysisIsolation. A datapoint contains all analysis results from a single instance and a single analysis. When a new analysis is done with the identical isolated DNA (AnalysisIsolation), a new DataPoint is made.

📄️ Analysis Overview

Analysis_Overview is not a table, but is an 'overview' screen used to put samples in a 96 well plate format.

📄️ Analysis Lysis Plate

Analysis Lysis Plate is the plate (layout) used for lysis, made using the Analysis Overview screen. The Analysis Lysis Plate record itself only contains basic information about the plate. The actual plate layout is stored in the Analysis Lysis records.

📄️ Analysis Isolation Plate

AnalysisIsolation Plate is the plate (layout) used for isolation, made using the Analysis Overview screen. The Analysis Isolation Plate record itself only contains basic information about the plate. The actual plate layout is stored in the Analysis Isolation records.

📄️ Analysis Plate

Analysis Plate is the plate (layout) used for Analysis. Similar to Analysis Isolation Plate, the record itself only contains basic information of the plate. The actual plate Layout is stored in the Analysis Datapoint records.
